cd "le répertoire"
sh NVIDIA-Linux-x86_64-367.27.run

le script blacklist nouveau ici :
/etc/modprobe.d/nvidia...conf

Rebooter, relancer le script si ca ne marche par : mkinitrd

Blacklister le driver nouveau :
echo "blacklist nouveau" >> /etc/modprobe.d/50-blacklist.conf && mkinitrd && reboot
